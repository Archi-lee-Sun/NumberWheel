A simple and visually appealing random number generator in the form of a spinning wheel.

✨ Features
- Interactive spinning wheel
- Custom range selection (min / max)
- Smooth animations
- Clean and modern UI

🎯 Purpose
This project was created as a learning experiment to explore:
- Frontend development (HTML, CSS, JavaScript)
- UI/UX design and user interaction
- Randomization logic

🤖 About the Development
The project was built with the help of AI tools (Claude), based on my own prompts, ideas, and design decisions.
